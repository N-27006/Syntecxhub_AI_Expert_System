# Rule-Based Expert System

## Objective
A simple AI-based expert system that diagnoses diseases based on symptoms using if-then rules.

## Features
- Takes user symptoms as input
- Applies rule-based reasoning
- Infers possible diseases
- Provides recommendations

## Rules Used
1. Fever + Cough → Flu
2. Flu → Consult Doctor
3. Fever + Headache → Viral Infection
4. Viral Infection → Take Rest

## How to Run

python main.py
